Percentage Change = new value - old value Original Value × 100
  • Step 1: Find the Original Value and New Value: Put them in the formula above

  • Step 2: Multiply by 100 Multiply the result by 100 to get a percentage

  • Step 3: Determine if it is an increase or decrease If the result is positive, there was an increase. If it is negative, there was a decrease.
